Math Today

Quadratic Equations

A quadratic equation is an equation of the form ax² + bx + c = 0 where a ≠ 0. The solutions can be found using the quadratic formula: x = (-b ± √(b²-4ac)) / 2a. The discriminant D = b²-4ac determines the nature of roots.

CS Yesterday

Binary Search Algorithm

Binary search finds the position of a target value in a sorted array. It compares the target to the middle element, then eliminates half the remaining elements. Time complexity: O(log n). Space complexity: O(1) iterative, O(log n) recursive.

Independence of Kazakhstan (1991)

Kazakhstan declared independence on December 16, 1991, following the dissolution of the Soviet Union. Nursultan Nazarbayev became the first president. The country joined the UN in March 1992 and adopted its current constitution in 1995.
